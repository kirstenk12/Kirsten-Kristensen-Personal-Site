JavaScript Knowledge



JavaScript can be written in three places: inline, internaly and externally. Our best option out of these is to use an external page and link it to your HTML page. To link these pages we use the < script > tag with an src="" attribute leading to your JavaScript code page.





Document Object Model (DOM):

JavaScript uses the DOM to manipulate and change elements of HTMl and CSS. DOM treats the HTML code as a tree diagram, displaying the relationships of the code.


Accessing elements:

We can access elements of the HTML code by ID, class name, element name and tag name using the syntax:
document.getElementByID("id")
document.getElementByClassName("className")
document.getElementByName()
document.getElementByTagName()
To change the elements we have selected we can add .innerHTML="", or .style="" to change the HTML and CSS properties respectively.


Assigning Variables:

We can assign variables using a few different strategies:
var variableName = n;
let variableName = n;
const variableName = n;
There is little difference between the var and let keywords, however let is most often used recently. On the other hand the const keyword makes sure that the variable cannot be changed later in the program.

Booleans
Null
and Undefined


JavaScript Operators:

+: addition
-: subtraction
*: multiplication
/: division
%: modulus(remainder)
++: increment
--: decrement
**: exponentiation
==: is equal to
===: is equal to (checks for both value and type)


Common HTML Events:

Onchange - when an HTML element has been changed

Onclick - when the user clicks an HTML element (click me to see an example)

Onmouseover - when the user moves the mouse over an HTML element
Onmouseout - when the user moves the mouse away from an HTML element (hover over me to see an example)

Onkeydown - when the user pushes a keyboard key

Onload - when the browser has finished loading the page


JavaScript Dialogs:

We have three different JavaScript dialogs:
Alert: information only dialog
Confirm: information and true or false dialog
and Prompt: information and text input dialog




JavaScript String:

Strings are declared using let, var or const ="string";
JavaScript strings are indexed starting from 0, however the string contents cannot be changed directly using indices.
JavaScript strings have a length property with a lot of useful methods. The length property can be accessed using the syntax: stringName.length;


Commonly Used String Methods:

endsWith()
indexOf()
lastIndexOf()
repeat()
replace()
search("")
slice(beginning, end) this method will return the items from the start position to the end position but will not alter the string or array.
splice(beginning, end) this method will take the items from the beginning to end, and remove them from the original string or array.
split("separator", n) this method will find the "separator" and take n items to the right.
startsWith()
toLowerCase() this method does not need any arguments.
toUpperCase() this method does not need any arguments.
toString this method does not need any arguments.
trim() this method does not need any arguments.
parseInt() or parseFloat() these methods can find a number at the start of a string and convert it into a number data type.


JavaScript Loops:

JavaScript supports the following loops:
for loop: ex. for(conditions){statements}
do while loop: ex. do{statements} while(conditions)
while loop: ex while(conditions){statements}
for in loop: ex. for(let i in x){console.log(i);} x being an array or a string, this loop will print the index numbers of x.
for of statement: ex. for(let c of s){console.log(c);} s being an array or a string, this loop will print the content of s.


Random Number Generator:

There is a function in JavaScript that can randomly generate numbers between your chosen range. The syntax for this function is math.random();




Commonly Used Array Methods:

Unlike strings, array elements can be modified/updated. The commonly used array methods are:
Concat: concatenates one or more arrays onto the receiver array
Find
findIndex
Includes
indexOf: indexOf(element) returns the first index at which element appears, or -1 if not found
lastIndexOf: lastIndexOf(element) returns the last index at which element appears, or -1 if not found
Join
Push: .push(element,…) adds one or more elements to the end of the array
Pop: .pop() removes and returns the last element of the array
Shift: .shift() removes and returns the first element of the array
unShift: .unshift(element, …) adds one or more elements to the front of the array
Reverse: reverse() reverses the elements of the array
Sort: sort() sorts the elements of the array in ascending order based on the start of the number (1, 10,100,2,200,3,4,5) this is how it would sort. To sort in numeric ascending order we have to use .sort(function(a,b){return a-b;})
Slice: slice(start, finish) returns a subarray beginning at start and ending just before finish and doesn’t remove the items from the original array.
Splice: splice(index, count(how many to remove), optionalValueWeWantToAdd) removes 'count' elements starting at index, and optionally adds new items.
